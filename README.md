# Rocket Lab (RKLB) Interactive Investment Thesis

This project presents an institutional-grade investment thesis on Rocket Lab (RKLB), developed for the **2026 Perplexity Computer Stock Pitch Competition**.

## Investment Recommendation

**LONG Rocket Lab (RKLB)**  
**Target Price:** $95  
**Expected Return:** +42%

## Core Insight

Rocket Lab is being priced like a rocket company, while over **67% of revenue comes from multi-year defense infrastructure contracts**, with approximately **$532M in contracted SDA revenue expected to convert in FY2027**, driving a visible EBITDA inflection.

## Features

This thesis includes:

- Interactive Discounted Cash Flow (DCF) Model  
- Monte Carlo Simulation (10,000 paths)  
- Scenario Analysis  
- Sensitivity Heatmap  
- Competitive Comparable Analysis  
- Risk Matrix Framework  
- Catalyst Timeline Visualization  

All financial outputs update dynamically based on user inputs.
